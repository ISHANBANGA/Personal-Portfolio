Desktop Application: Cricket Knockdown Tournament

The Cricket Knockdown Tournament application is a desktop software developed using C# Windows Forms. It simulates a cricket tournament where teams are randomly distributed to compete against each other. The initial matchups lead to semi-final rounds based on the scores achieved in the preliminary matches. The winning teams advance to the final round where they compete for the championship. This application incorporates random team distribution, score tracking, and progression logic to mimic the dynamics of a real cricket tournament. The intuitive interface allows users to easily interact with the software, manage games, and view results in real time, culminating in a climactic final match where the champion is crowned.

Key challenges included scalability, optimizing database management for performance, and ensuring consistent functionality across platforms. This project enhanced my skills in ASP.NET MVC, SQL, and full-scale web application development.
